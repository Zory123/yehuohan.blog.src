基类指针可访问的内存地址长度，比派生类指针可访问的内存地址长度要短。
C++中可以用基类指针指向派生类实例，但base_ptr可访问的地址长度只限于BaseClass范围，如下图所示；反之，用派生类指针指向基类实例则是**不允许**的，因为derived_ptr访问BaseClass范围之外的地址时，则会发生指针越界

```text

base_ptr |---------------|---------------| derived_ptr
      |  |               |               |   |
      |  |Base class     | Derived Class |   |
      |  |               |               |   |
      -  |---------------|               |   |
                         |               |   |
                         |               |   |
                         |               |   |
                         |---------------|   -

```
